## Link customizado

Para el caso de usar react-router o gatsby donde se usa una etiqueta `Link` para poder moverse entre vistas.

> Tanto el children y el className se usan de forma obligatoria, el children sirve para poder renderear el contenido del Card
> y el className para poder darle estilos con styled-componentes

<Playground>
  <Card
    renderLink={({ children, className }) => (
      <a href="/example" className={className}>{children}</a>
    )}
    author="Wilson Flores"
    avatar="https://avatars0.githubusercontent.com/u/4754339?s=400&v=4"
    date="10 junio, 2017"
    title="Titulo del post"
    tags={["typescript", "javascript"]}
  />
</Playground>
